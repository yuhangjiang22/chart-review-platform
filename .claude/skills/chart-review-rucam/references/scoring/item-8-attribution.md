# Item 8 — Clinician Attribution (not scored; required for adjudication)

**Goal:** Capture explicit clinician statements about the drug–liver injury causal relationship.
This is NOT part of the 7-item total score but is required for every case.

### Search notes for:

**Positive attribution:**
- "Drug-induced liver injury" / "DILI" attributed to suspect drug
- "Likely/probable/possible due to [drug]"
- "Concern for [drug]-induced hepatitis/cholestasis"
- "Stopped [drug] due to LFT elevation / suspected DILI"
- Rechallenge language: "restarted and LFTs rose again"

**Negating statements:**
- "DILI unlikely"
- "LFTs explained by ischemia/sepsis/obstruction"
- "Continue drug — not suspected"
- Alternative diagnosis explicitly named as the cause

### Record for each statement:
- Verbatim quote (or close paraphrase)
- Note type (hepatology consult / discharge summary / ED note / progress note)
- Note date and DAYS_FROM_LIVER_INJURY

### Common mistakes
- Skipping this step because items 1–7 are complete: attribution is required even when the RUCAM score is clear.
- Only capturing positive attribution: negating statements are equally important.
