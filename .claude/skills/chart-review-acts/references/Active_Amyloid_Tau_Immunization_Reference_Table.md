# Active Amyloid / Tau Immunization Reference Table

Companion to the [Vaccine Extraction Guideline](../guidelines/Vaccine_Extraction_Guideline.md), covering the **`Active Amyloid or Tau Immunization`** category — the investigational Alzheimer's-disease vaccines that are **not** in the [CDC vaccine reference table](CDC_Vaccine_Reference_Table.md). Source: Alzforum Therapeutics (filter: Immunotherapy-active, condition = Alzheimer's Disease).

## How the mapping agent should use this table

- Match an extracted vaccine/therapy name (or its development code / alias) against the **Therapy name** and **Aliases** columns. A match → category `Active Amyloid or Tau Immunization` — **regardless of development status** (discontinued/terminated candidates still count, since they appear in historical and trial documentation).
- These are matched by **name/code only**; they are not derived from the CDC table and are never `Live` / `Non-Live`.
- **Exclude passive monoclonal antibodies** (e.g., lecanemab/Leqembi, donanemab/Kisunla, aducanumab/Aduhelm, gantenerumab, solanezumab, crenezumab, bapineuzumab, remternetug) → these map to `Not a vaccine`.
- **Exclude non-antigen "active immunotherapy" agents** (adjuvants / immunomodulators such as CpG 1018, AD04/alum, Protollin; and the TERT peptide GV1001) — they are not Aβ/tau immunizations (see Notes).

## Notes

- **Active vs. passive (scope of this table):** This table contains only **active immunizations / vaccines**, where an antigen (an Aβ or tau peptide/fragment/mimotope/DNA/liposome construct) is administered to stimulate the patient's own immune system to generate antibodies. **Passive immunotherapy monoclonal antibodies are NOT vaccines and are deliberately excluded** — these include aducanumab (Aduhelm), lecanemab (Leqembi), donanemab (Kisunla), gantenerumab, solanezumab, crenezumab, bapineuzumab, remternetug, and similar anti-Aβ/anti-tau mAbs. Downstream, those passive antibodies should map to **"Not a vaccine."** (Note: Alzinova's ALZ-201 is a monoclonal antibody derived from the ALZ-101 vaccine — ALZ-201 would be passive/excluded; ALZ-101 the vaccine is included here.)
- **Amyloid vs. tau split:** Most AD active vaccines target **amyloid-beta (Aβ)** (AN-1792, CAD106/amilomotide, ACC-001, UB-311, ABvac40, ACI-24.060, ALZ-101, AV-1959, Lu AF20513, Affitope AD01/AD02/AD03, V-950). A smaller set targets **tau** (AADvac1, JNJ-64042056/ACI-35.030, AV-1980R/A). **DUVAX** is a dual Aβ+tau vaccine. The Nuravax/Institute for Molecular Medicine candidates (AV-1959, AV-1980, DUVAX) all share the **MultiTEP** platform.
- **Excluded "active immunotherapy" entries that are NOT amyloid/tau vaccines.** The Alzforum "Immunotherapy (active)" filter returns several items that are *not* antigen-specific Aβ/tau immunizations and were excluded from this table:
  - **GV1001** (tertomotide / RIAVAX) — a telomerase (TERT)-derived peptide acting via HSP70/HSP90 and anti-inflammatory pathways, not an Aβ/tau immunization (Phase 3 in AD). Excluded.
  - **CpG 1018** (CpG ODN; Dynavax) — a TLR9-agonist **adjuvant/immunomodulator**, not an antigen-containing vaccine (Phase 1 in AD). Excluded.
  - **Protollin** (I-Mab / Jiangsu Nhwa) — a nasal innate-immune stimulant (bacterial outer-membrane proteins + LPS acting via TLRs); promotes monocyte-mediated Aβ clearance but is not an active Aβ/tau immunization (Phase 1). Excluded.
  - **AD04** (Alhydrogel / alum) — an **aluminum-hydroxide adjuvant** being repurposed as monotherapy; contains no Aβ/tau antigen, target listed as "Unknown" (Phase 2). It surfaced as the placebo comparator in the Affitope AD02 program. Excluded.
- **Could not fully verify from Alzforum:** Affitope AD01/AD02/AD03 and V-950/Merck details were confirmed via web search and literature (Alzforum has an Affitope AD02 profile) rather than fully rendered Alzforum profile pages; statuses (all discontinued) are consistent across sources. The AV-1959R/AV-1980R/DUVAX rows are listed on the Alzforum active-immunotherapy results but their individual profile pages returned limited detail; sponsor is given on Alzforum as Nuravax (the AV-1959D profile cited the Institute for Molecular Medicine as trial registrant — same MultiTEP program). Historical full-length-Aβ academic vaccines and very early-stage/preclinical constructs were not exhaustively enumerated.
